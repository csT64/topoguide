# Polices personnalisées

Placer ici les fichiers de police propriétaires utilisés pour la génération des PDF :

- `futuramediumbt.ttf`  — corps du texte (FuturaMediumBT)
- `FuturaHeavyfont.ttf` — titres (Futura Heavy)

Ces fichiers ne sont pas versionnés (propriétaires).
En leur absence, `TopoguideService` utilise automatiquement la police `helvetica` (intégrée à TCPDF).
